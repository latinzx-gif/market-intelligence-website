# Website Public Assets

## Purpose

`/Website_Content/Assets` is the approved holding area for public-safe images and visual references used by the market intelligence website.

## Allowed Assets

- Public-safe portfolio visuals
- Approved case study preview images
- Public-ready insight card graphics
- Non-confidential packaging comparison visuals
- Non-confidential social share images

## Do Not Store

- Private client screenshots
- Raw review exports
- Confidential shelf photos
- Private Obsidian attachments
- Unapproved product images
- Any asset without usage permission or clear public-safe status

## Web Deployment Note

For browser rendering, copy approved deployable image files into `market-intelligence-website/public` and reference them from markdown with a public path such as `/assets/example.png`.
