# DataClaw V2 Design System

## Core Identity
**Goal:** Premium Intelligence Platform
**Vibe:** Authoritative, clean, data-dense but breathable (CB Insights, Mintel, Stripe).

## Color Palette
- **Background:** `#FFFFFF` (Clean, editorial canvas)
- **Text (Primary):** `#111827` (High-contrast, authoritative dark slate)
- **Muted (Borders, Secondary Text):** `#CBD5E1` (Subtle, non-distracting structural lines)
- **Accent:** `#22C55E` (Vibrant green for positive signals, primary actions, and "Verified" indicators)

## Geometry & Spacing
- **Border Radius:** `20px` (Modern, approachable, premium card feel)
- **Shadows:** `soft` (Subtle elevation, mimicking physical premium paper/cards)
- **Grid:** 12-column fluid grid
- **Spacing:** `large` (Generous padding to let data breathe)

## Typography
- **Style:** Editorial (Serif for headings to convey authority/publishing, crisp Sans-serif for data/UI elements).
- **Hierarchy:** Strict. Clear distinction between eyebrow text, H1, H2, and body copy.

## Data Visualization Rules
- **Allowed:** Timelines, Heatmaps, Bubble Charts, Matrices.
- **Forbidden:** Pie Charts.
- **Style:** Integrated into the UI, not floating images. Use the accent color strategically.

## Interaction & Motion
- Hover expansions for dense data cards.
- Sticky navigation for deep reports.
- Scroll progress indicators on long-form intelligence.
- Smooth, non-jarring transitions (150ms-300ms ease-in-out).
